API Gestion des bénéficiaires effectifs

Mon rôle : Réalisé en autonomie dans le cadre d’un challenge technique, ce projet m’a permis de structurer une API complète avec modélisation métier et logique récursive.

1. Acceuil

Cette API REST développée avec Java 21 et Spring Boot permet de déterminer les bénéficiaires effectifs d'une entreprise, en conformité avec la réglementation KYC (Know Your Customer). Elle identifie automatiquement toute personne physique détenant plus de 25 % du capital d'une entreprise, directement ou indirectement.

2. Objectif principal

L'objectif du MVP est de permettre, via une seule route REST GET, de récupérer la liste des bénéficiaires effectifs d’une entreprise donnée. Cette fonctionnalité est essentielle pour les organismes financiers dans le cadre de leurs obligations de conformité.

3. Fonctionnalité développée (Must Have)

  • Route GET pour récupérer les bénéficiaires effectifs d'une entreprise
  • Prend en entrée : ID de l’entreprise + filtre sur le type de bénéficiaires
  • Retourne :
    • 200 : liste des bénéficiaires avec pourcentages de détention
    • 204 : aucun bénéficiaire trouvé
    • 404 : entreprise inexistante

4. Mise en œuvre technique

  • Modélisation métier (entreprise, personne physique, liens de détention)
  • Calcul récursif des détentions indirectes pour identifier les >25%
  • Utilisation de la Clean Architecture (hexagone)
  • Tests unitaires automatisés sur la logique de calcul
  • Stockage des données en mémoire (pas de base de données)

5. Résultat

L’API fonctionne selon les spécifications : la route GET répond correctement selon les différents cas d’usage. La logique de calcul des bénéficiaires est conforme aux exigences réglementaires et aux règles métier définies.

6. Apports du projet

Ce projet m’a permis d’approfondir la modélisation métier dans un domaine réglementé, d’implémenter un algorithme de calcul complexe et de structurer une API propre selon les principes DDD et hexagonaux. Il illustre ma capacité à livrer une fonctionnalité métier complète de manière autonome dans un temps restreint.
